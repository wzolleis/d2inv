# d2inv
Division 2 Inventory

Startet den Server mit dem lokalen Profil: -Dspring.profiles.active=local

* Startet den Client (Dev): npm start
* Aufruf der UI im Server
* API: http://localhost:8080/api
* UI: http://localhost:8080/ui
